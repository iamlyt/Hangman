# Hangman

## Overview
Hangman is a popular yet grim puzzle game. A cruel computer hides a word from you, which you try to guess letter by letter. If you fail, you'll be “hanged”. If you win, you'll survive.

## Topics covered 
*but not limited to...* 
- split() and join()
- lists
- fstring
- variables
- sets
- tuples
- slicing
